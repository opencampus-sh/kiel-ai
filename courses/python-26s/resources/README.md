# Resources

Here are some resources, that might come in handy:

* The [Harvard Course](harvard-course.md) is a detailed, in depth lecture on Python programming, take a look there if you want to have a deep dive. I provided little programming challenges for each chapter, that you can use for practice
* The [Worklabs](worklabs.md) are the notebooks we discussed in class, take a look again to see what we did
* A useful resource might be [W3Schools.com](https://www.w3schools.com/python/) for quick reference on how to use Python features. They also have an Online Playground to run Python code
* [Think Python](https://greenteapress.com/wp/think-python-3rd-edition/) is a textbook directed at absolute beginners. You can read it online for free. The [second edition](https://greenteapress.com/wp/think-python-2e/) can be downloaded as a pdf.
